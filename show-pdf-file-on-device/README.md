端末のネットワーク隔離をしたタイミングで、端末上に任意のPDFファイルを配置、端末上に表示させるためのスクリプトです。

（ファイルの配置はRTRで別途putコマンドで実施するか、FusionワークフローのPut Fileアクションを利用してください）
